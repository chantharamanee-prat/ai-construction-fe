I'm developing a PoC to assess the feasibility of using a YOLO machine learning model for construction progress monitoring. The chosen framework is Ultralytics. I have a collection of raw images that I will use to train the model to output a completion percentage. It is important to note that the current dataset only contains images of projects up to 50% completion, which is acceptable for the purpose of this PoC. The Ultralytics library is already configured in a virtual environment within the project. My next steps are to focus on data preparation for the training phase and then evaluate the performance of the final trained model.
